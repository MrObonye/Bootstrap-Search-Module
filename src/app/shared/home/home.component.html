Bootstrap Search Module

This is a simple search module. Two search widgets are included with this module. This module uses bootstrap for the styles.

Widget 1 as seen in header

Widget 2 as seen in footer

Dependency

Bootstrap 4

How to set up module in application

Step 1. Bring in the search module folder into your Angular project.

Step 2. Add SearchModule to your app.module imports.

Step 3. In the searchService add the api endpoint.

Step 4. Update the SearchOption interface for what you are expecting from the API.

Step 5. Add widgets throughout your application.
